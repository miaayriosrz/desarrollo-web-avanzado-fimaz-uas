// Objetivo //
Crear un sistema completo que use Herencia, Polimorfismo y Excepciones para gestionar diferentes tipos de usuarios (Admin, Alumno, Invitado) y mostrarlos en una tabla.

// Requisitos // 
 Servidor local: XAMPP (Apache activo).

PHP 8.0 o superior

// Evidencia // 
El sistema intenta registrar un cuarto usuario con un correo inválido. Gracias al bloque try-catch, la página no se rompe, sino que muestra un aviso en color rojo explicando el error, mientras que los usuarios anteriores se imprimen correctamente en la tabla.
